# CINEMAITE
</div>

## **How to install and run locally ?**

```
$ Install Git & Node (if not installed)
$ git clone https://github.com/Vishruta-Patil/Cinemaite
$ npm install
$ npm start
$ Server is running on localhost:8080
```
## **Features -**

- Movie List By year
- Filter Movies By Genres
- Search Movies
- Infinite Scroll 
- Starts with the year 2012
- Responsive UI
 

## **Built with -**

- TypeScript
- React JS
